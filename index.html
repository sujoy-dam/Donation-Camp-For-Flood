Important FAQ ?

What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming interface for web documents, allowing structured access and manipulation of HTML or XML elements using scripting languages like JavaScript.

How do you select an element from the DOM ?

To select an element from the DOM, you can use methods like getElementById, querySelector, or getElementsByClassName to access and manipulate specific elements.

What is event delegation in the context of the DOM, and why is it useful ?

Event delegation is a technique where a parent element handles events for its child elements. It improves performance by reducing event listeners and managing dynamic elements efficiently.

How do you manipulate an element's attributes and styles using the DOM ?

You manipulate an element's attributes using setAttribute, getAttribute, and directly change styles via the style property, e.g., element.style.color = "red".
